# 8. 基础设施即代码 (IaC)

## 学习目标
掌握基础设施即代码工具，能够自动化基础设施的配置和管理。

## 核心内容
### IaC 概念
- 声明式 vs 命令式
- 状态管理
- 版本控制

### Terraform
- HCL 语法
- Provider 和 Resource
- 模块化
- 状态管理
- 工作空间

### 配置管理
- **Ansible** - 无代理配置管理
- **Puppet** - 声明式配置管理
- **Chef** - 基于 Ruby 的配置管理
