# 5. 服务器管理

## 学习目标
掌握服务器管理技能，包括 Web 服务器配置、负载均衡和缓存。

## 核心内容
### Web 服务器
- **Nginx** - 高性能反向代理和 Web 服务器
- **Apache** - 传统 Web 服务器
- **IIS** - Windows 服务器

### 代理和负载均衡
- 正向代理 vs 反向代理
- 负载均衡策略
- CDN 边缘服务器

### 缓存
- 缓存服务器概念
- Redis 缓存
- Memcached
