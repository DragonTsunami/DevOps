# 12. 软件工程实践

## 学习目标
掌握软件开发生命周期和敏捷开发方法，能够高效协作。

## 核心内容
### 软件开发生命周期 (SDLC)
- 需求分析
- 设计
- 开发
- 测试
- 部署
- 维护

### 敏捷开发
- **Scrum** - 最流行的敏捷框架
- **Kanban** - 看板方法
- **SAFe** - 规模化敏捷框架

### 测试
- 单元测试
- 集成测试
- 端到端测试
- 自动化测试
