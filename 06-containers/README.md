# 6. 容器技术

## 学习目标
掌握 Docker 容器技术，能够构建、运行和管理容器化应用。

## 核心内容
### Docker 基础
- 容器概念
- Docker 镜像和容器
- Dockerfile 编写
- Docker 命令

### Docker Compose
- 多容器应用编排
- YAML 配置文件
- 服务定义和依赖

### 高级主题
- Docker 网络
- 数据卷
- 镜像优化
- 安全最佳实践
