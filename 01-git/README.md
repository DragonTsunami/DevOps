# 1. Git

## 学习目标
掌握版本控制系统 Git 的使用，包括基本命令、分支管理和团队协作。

## 核心内容
- Git 基本命令（clone, add, commit, push, pull）
- 分支管理（branch, checkout, merge）
- 解决冲突
- Pull Request 工作流
- Git 平台使用（GitHub, GitLab）

## 推荐资源
- [Pro Git Book](https://git-scm.com/book/en/v2) - 免费电子书
- [Learn Git Branching](https://learngitbranching.js.org/) - 交互式学习
- [Atlassian Git 教程](https://www.atlassian.com/git)
- [Git Command Explorer](https://gitexplorer.com/)

## 实践项目
创建一个 Git 仓库，练习分支管理和 Pull Request 流程。
