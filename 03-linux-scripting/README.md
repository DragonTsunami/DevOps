# 3. Linux & 脚本

## 学习目标
掌握 Linux 操作系统和 Shell 脚本编程，能够进行服务器管理和任务自动化。

## 核心内容
### Linux 基础
- 常用命令：ls, cd, mkdir, rm, cp, mv
- 文件权限管理：chmod, chown
- 进程管理：ps, kill, top
- 系统信息：df, du, free
- 网络工具：ssh, scp, wget, curl

### Shell 脚本
- Bash 基础语法
- 变量和参数
- 流程控制
- 函数定义
- 文本处理（grep, sed, awk）

## 推荐资源
- [Shell Scripting Tutorial](https://www.shellscript.sh/) - 免费
- [Linux Command Handbook](https://www.freecodecamp.org/news/the-linux-commands-handbook/)
- [Bash Reference Manual](https://www.gnu.org/savannah-checkouts/gnu/bash/manual/bash.html)
- [Ubuntu 入门指南](https://itsfoss.com/getting-started-with-ubuntu/)

编写一个 Bash 脚本，实现系统监控和日志分析。
