# 9. CI/CD 持续集成/持续部署

## 学习目标
掌握 CI/CD 流水线，能够自动化构建、测试和部署流程。

## 核心内容
### CI/CD 概念
- 持续集成 (CI)
- 持续部署 (CD)
- 流水线阶段

### 工具
- **Jenkins** - 开源自动化服务器
- **GitHub Actions** - GitHub 原生 CI/CD
- **GitLab CI/CD** - GitLab 内置
- **Azure DevOps** - 微软 DevOps 平台

### 最佳实践
- 自动化测试
- 环境管理
- 密钥管理
- 部署策略
