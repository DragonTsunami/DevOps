# 11. 云平台

## 学习目标
掌握至少一种主流云平台，能够部署和管理云上资源。

## 核心内容
### 主流云平台
- **AWS** - 市场份额最大
- **Azure** - 微软云平台
- **GCP** - 谷歌云平台

### 核心服务
- 计算服务（EC2, VM, Compute Engine）
- 存储服务（S3, Blob Storage, Cloud Storage）
- 网络服务（VPC, Virtual Network）
- 数据库服务（RDS, SQL Database）

### 认证
- AWS Certified Cloud Practitioner
- Azure Fundamentals (AZ-900)
- Google Cloud Associate Cloud Engineer
