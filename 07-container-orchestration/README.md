# 7. 容器编排

## 学习目标
掌握 Kubernetes 容器编排技术，能够部署和管理容器化应用。

## 核心内容
### Kubernetes 基础
- 集群架构
- Master 和 Worker 节点
- Pod、Deployment、Service
- ConfigMap、Secret

### 高级概念
- Ingress 控制器
- 持久化存储（PV、PVC）
- StatefulSet、DaemonSet
- Job、CronJob

### 工具
- kubectl 命令行工具
- Helm 包管理器
- 监控和日志
