# DevOps 学习路径

这是一个基于 [DevOps-Roadmap](https://github.com/milanm/DevOps-Roadmap) 的完整学习路径，帮助你从零开始成为 DevOps 工程师。

## 学习路径

### 基础阶段
1. **[Git](01-git/)** - 版本控制系统
2. **[编程语言](02-programming/)** - Python/Go/JavaScript
3. **[Linux & 脚本](03-linux-scripting/)** - 操作系统和自动化
4. **[网络与安全](04-networking-security/)** - 网络基础和安全

### 进阶阶段
5. **[服务器管理](05-server-management/)** - Web 服务器和代理
6. **[容器技术](06-containers/)** - Docker 和容器化
7. **[容器编排](07-container-orchestration/)** - Kubernetes
8. **[基础设施即代码](08-iac/)** - Terraform/Ansible

### 高级阶段
9. **[CI/CD](09-cicd/)** - 持续集成/持续部署
10. **[监控与可观测性](10-monitoring/)** - Prometheus/Grafana
11. **[云平台](11-cloud/)** - AWS/Azure/GCP
12. **[软件工程实践](12-software-engineering/)** - Scrum/SDLC
13. **[DevSecOps](13-devsecops/)** - 安全集成 (Bonus)

## 学习建议

1. **按顺序学习** - 每个阶段都建立在前一个阶段的基础上
2. **动手实践** - 每个主题都有实践项目，务必完成
3. **建立作品集** - 将你的项目上传到 GitHub
4. **持续学习** - DevOps 领域发展迅速，保持学习
