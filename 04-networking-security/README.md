# 4. 网络与安全

## 学习目标
理解网络基础和安全概念，能够配置和管理网络服务。

## 核心内容
### 网络基础
- OSI 七层模型
- TCP/IP 协议
- DNS 工作原理
- HTTP/HTTPS
- 防火墙配置

### 安全概念
- 密钥管理
- SSL/TLS 证书
- 网络安全策略
- 访问控制
